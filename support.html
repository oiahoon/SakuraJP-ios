🔧 故障排除

应用无法启动或闪退怎么办？

请尝试以下解决方案：

  1. 强制关闭应用并重新启动
  2. 重启您的设备
  3. 检查应用是否有更新版本
  4. 确保iOS系统版本支持（需要iOS 13.0或更高版本）
  5. 如果问题持续，请联系技术支持
AI辅导功能无法使用？

AI功能需要网络连接，请检查：

  1. 设备是否连接到互联网
  2. 网络连接是否稳定
  3. 是否达到每日使用限制（免费用户每日10次）
  4. 在设置中检查AI配置是否正确
学习数据丢失了怎么办？

数据恢复步骤：

  1. 检查是否登录了相同的Apple ID
  2. 确保iCloud备份功能已开启
  3. 尝试重新安装应用（数据可能会自动恢复）
  4. 如果数据无法恢复，请联系技术支持
语音功能无法正常工作？

请检查以下设置：

  1. 确保设备音量已开启
  2. 检查应用的麦克风权限
  3. 在设置中选择合适的TTS引擎
  4. 确保网络连接正常（Google TTS需要网络）
订阅或付费相关问题？

订阅管理：

  1. 所有订阅通过App Store管理
  2. 在iPhone设置 > Apple ID > 订阅中查看
  3. 可以随时取消或修改订阅
  4. 退款请求需要通过Apple客服处理
